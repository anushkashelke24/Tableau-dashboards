## Overview
This dashboard analyzes Airbnb listings to understand pricing, demand, and revenue trends.

## Key Insights
- Average price increases with number of bedrooms
- Listing distribution by bedroom count
- Price comparison across zip codes
- Revenue trend over time (weekly)

## Visuals Included
- Avg Price per Bedroom 
- Price per Zipcode 
- Revenue Trend by Week 

## Tools Used
- Tableau Public
- CSV Dataset (Airbnb listings)

## Files
- Airbnb_Dashboard.twbx 

## How to View
Download the `.twbx` file and open it using **Tableau Public** or **Tableau Desktop**.
